# SecureCRT-Keyword-Highlighting
SecureCRT Keyword Highlighting for OS/Network Administrator

## How to use ?
- Open "My Computer" --> %USERPROFILE%\AppData\Roaming\VanDyke\Config
- Create Folder "Keywords"
- Download the file "Keywork-OS.ini" or "Keywork-Network.ini" to Folder "Keywords"
- Open SecureCRT, Options > Edit Default session > Keyword Highlighting > List Name > then select “Keywork-OS” from drop down
